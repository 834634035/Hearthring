# Stylized Nature MegaKit Standard 模型目录

这份列表对应 `manifest.json` 中 `pack` 为 `stylized-nature-megakit-standard` 的环境模型。`模型 ID` 可以直接用于项目里的模型加载逻辑，`用途` 用来快速判断它适合放在什么场景里。

## 总览

| 分类 | 数量 | 用途 |
| --- | ---: | --- |
| bushes | 2 | 灌木、林地边缘、村落外围遮挡 |
| flowers | 9 | 花丛、花瓣、地表装饰、祭坛周边点缀 |
| ground-cover | 11 | 草、蕨、三叶草、小型植物，用于铺地和填空 |
| mushrooms | 2 | 林下、潮湿区域、荒野细节 |
| rocks | 24 | 石头、碎石、石径、道路边缘、山地或营地地表 |
| trees | 20 | 普通树、枯树、松树、扭曲树，用于不同生态区域 |

## 快速组合建议

| 场景 | 推荐组合 |
| --- | --- |
| 普通森林 | `commonTree*` + `bushCommon` + `grassCommonShort` + `fern1` |
| 神秘祭祀林地 | `twistedTree*` + `mushroomLaetiporus` + `petal*` + `flower*` |
| 荒地或危险区域 | `deadTree*` + `rockMedium*` + `grassWispyTall` + `pebbleSquare*` |
| 山地松林 | `pine*` + `rockMedium*` + `grassWispyShort` + `pebbleRound*` |
| 村落小径 | `rockPathSquare*` 或 `rockPathRound*` + `flower*` + `bushCommonFlowers` |
